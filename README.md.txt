# Digital Locker System

A Java-based GUI project that allows users to securely register, log in, and manage their personal files (upload, download, delete). 

## Features

- **User Registration & Login**
- **Secure File Uploading**
- **Download & Delete Files**
- **Change Password**
- **Dark Themed Modern UI**
- **Local File Storage (per user)**

## Tech Stack

- **Java (Swing)**
- **MySQL** for user authentication
- **JDBC** for database interaction
- **GitHub** for version control

## Folder Structure

/src /com/digitallocker /view /dao /util /user_files

## How to Run

1. Install **MySQL** and set up the `users` table.
2. Update your database credentials in `DatabaseConnection.java`
3. Run the project using any Java IDE (like IntelliJ or Eclipse)
